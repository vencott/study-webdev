# 문서의 구조와 슈퍼스타들

## HTML 문서의 구조를 이루는 태그들

```
<!DOCTYPE html> // 관례적으로 표시, 이 문서는 html 문서이다
<html> // html 문서의 최상위 태그
<head> // 문서의 설명
    <meta charset="UTF-8"> // UTF-8 형식으로 문서를 열어라
    <title>Web1 - html</title> // HTML 문서의 제목
</head>

<body> // 본문

</body>
</html>
```

- 거의 모든 웹 사이트들이 위와 같은 형식을 갖추고 있다
